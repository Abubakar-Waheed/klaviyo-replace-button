# Privacy Policy

Klaviyo Replace Button Restorer does not collect, store, sell, transfer, or share user data.

The extension runs only on Klaviyo pages and only changes the visibility of the existing image Replace button in the Klaviyo email template editor. It does not transmit data to any server, use analytics, track browsing activity, store credentials, read messages, or collect website content.

The extension does not use remote code. All code is included in the extension package.

For support, visit:

https://github.com/Abubakar-Waheed/klaviyo-replace-button/issues
